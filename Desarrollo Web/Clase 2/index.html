Intruciones paso a paso Semipermente

  1. Preparar uñas: LImpiar, empujar cuticulas y limar.
  2. Limpiar superficie: Usar limpiador o alcohol para eliminar residuos.
  3. Aplicar base coat: Cubrir la uña, sellar bordes y curar con lampara UV/LED.
  4. Aplicar color: Pintar en capas finas, sellar bordes y curar con lampara.
  5. Aplicar top coat: Sellar y curar nuavamente
  6. Eliminar capa pegajosa: Usar limpiador o alcohol.
  7. HIdratar: Aplicar aceite para cuticulas.
